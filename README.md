# AI Job Insight
AI-powered tool to analyze freelance jobs and generate better proposals.

クラウドソーシング案件の応募価値を分析し、応募判断から投稿文作成までを支援するAIツール

## 概要

AI Job Insightは、クラウドソーシングサイト（クラウドワークスなど）の案件情報とあなたのプロフィール情報をもとに、その案件に応募する価値があるかどうかをAIが分析し、応募判断の参考情報を提供するツールです。

### 主な機能

- **案件価値の判断** - AIが案件とプロフィールの適合度を分析
- **応募戦略の検討** - 強調すべきスキルや提案ポイントをアドバイス
- **応募投稿文の作成支援** - AIとの対話で投稿文を段階的にブラッシュアップ
- **投稿文テンプレートの管理** - 再利用可能なテンプレートとして保存・管理
- **履歴管理** - 過去の分析結果を保存・参照
- **応募予定案件管理** - 応募予定の案件をリストで管理

---

## 特徴

### ユーザー自身のAIを利用

本システムは、ユーザー自身が設定した生成AIサービス（OpenAI、Google Gemini）を利用します。

- サーバー側でAI処理は行わない
- APIキーはブラウザ内でのみ利用
- APIキーをサーバーへ送信しない

### 完全クライアント型（v1）

- Webブラウザ上で動作
- バックエンドサーバー不要
- データはブラウザ内ストレージに保存

### AI提案承認方式

AIが生成または改善した内容は、以下の手順で反映します。

1. AI提案を表示
2. ユーザー確認
3. 承認ボタン押下
4. データ反映

AIは保存データを自動で書き換えません。

## 利用方法

### 1. APIキーの設定

使用するAIサービス（OpenAI または Google Gemini）を選択し、APIキーを設定します。

#### OpenAI APIキー取得方法

1. [OpenAI公式サイト](https://openai.com/)でアカウント作成
2. [API Keysページ](https://platform.openai.com/api-keys)へアクセス
3. Create new secret keyボタンを押下
4. 表示されたAPIキーをコピー

#### Google Gemini APIキー取得方法

1. [Google AI Studio](https://aistudio.google.com/)へアクセス
2. Googleアカウントでログイン
3. Get API Keyボタンを押下
4. 表示されたAPIキーをコピー

### 2. プロフィール情報の入力

以下の情報を入力します。

- 自己紹介
- スキル
- 実績
- 得意分野
- スキル項目ラベル（任意）

### 3. 案件情報の入力

分析したい案件の情報を入力します。

- 案件説明
- 仕事内容
- 条件
- 報酬
- 案件URL（任意）
- 案件メモ（任意）

### 4. 分析実行

分析実行ボタンを押下すると、AIが案件を分析し、以下の情報を提供します。

- **応募おすすめ度** - ★1〜★5の5段階評価
- **案件評価理由** - 良い点と注意点
- **スキル適合度** - レーダーチャート表示
- **応募戦略** - 強調すべきスキルと提案ポイント

### 5. 投稿文作成（オプション）

分析結果をもとに、AIと対話しながら応募用投稿文を作成できます。

#### 質問・相談モード

案件内容や分析結果について質問・相談できます。

#### 投稿文作成モード

追加情報を入力しながら、投稿文を段階的に作成・改善できます。

- 既存テンプレートを使用
- 新規作成

## パフォーマンス

## 制約事項

- v1ではバックエンドおよびデータベースを必須としない
- データ保存はブラウザ内ストレージのみ
- 保存データは同一ブラウザ・同一端末内でのみ参照可能
- APIレート制限はAIサービス側の仕様に依存
