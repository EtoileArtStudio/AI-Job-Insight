## 利用方法ガイド内容

### 1. アプリの概要説明

#### AI Job Insightとは

クラウドソーシングサイトの案件情報とあなたのプロフィール情報をもとに、その案件に応募する価値があるかどうかをAIが分析し、応募判断の参考情報を提供するツールです。

#### 主な機能

- 案件価値の判断
- 応募戦略の検討
- 応募投稿文の作成支援
- 投稿文テンプレートの管理
- AIとの対話による応募内容のブラッシュアップ

### 2. 案件分析の手順

#### 手順1：APIキーの設定

1. 使用するAIサービス（OpenAI または Google Gemini）を選択
   - サービス切替時に自動で推奨モデルが設定されます
2. APIキーを入力（任意）
   - 空の場合はデモモードで動作します
3. モデル名を選択または入力（任意）
   - 候補リストから選択、または直接入力が可能
   - OpenAIの推奨モデル：gpt-4o-mini
   - Geminiの推奨モデル：gemini-1.5-flash
   - モデル一覧は公式ドキュメントリンクから確認できます
4. 保存ボタンを押下

#### 手順2：プロフィール情報の入力

1. 自己紹介を入力
2. スキルを入力
3. 実績を入力
4. 得意分野を入力
5. スキル項目ラベルを設定（任意）

#### 手順3：案件情報の入力

1. 案件説明を入力
2. 仕事内容を入力
3. 条件を入力
4. 報酬を入力
5. 案件URLを入力（任意）
6. 案件メモを入力（任意）

#### 手順4：分析実行

1. 分析実行ボタンを押下
2. 分析結果を確認

### 3. 投稿文作成方法

#### 質問・相談モード

1. 分析結果表示後、対話支援領域で「質問・相談モード」を選択
2. 案件内容や分析結果について質問を入力
3. AIの回答を確認

#### 投稿文作成モード

1. 分析結果表示後、対話支援領域で「投稿文作成モード」を選択
2. 投稿文作成方式を選択
   - 既存テンプレート使用
   - 新規作成
3. 追加情報を入力（エントリー人数、他応募者の投稿傾向など）
4. AIと対話しながら投稿文を作成
5. 投稿文保存ボタンを押下

### 4. テンプレート利用方法

#### テンプレート保存

1. 投稿文作成後、テンプレート化ボタンを押下
2. テンプレート保存方式を選択
   - AIによる成形処理
   - 手作業で編集した内容をそのまま保存
3. テンプレート名を入力
4. 保存

#### テンプレート使用

1. 投稿文作成モードで「既存テンプレート使用」を選択
2. テンプレート一覧から使用するテンプレートを選択
3. AIと対話しながら内容を調整

#### テンプレートブラッシュアップ

1. テンプレート一覧からブラッシュアップしたいテンプレートを選択
2. テンプレートブラッシュアップボタンを押下
3. AIへ改善依頼を入力
4. AI提案内容を確認
5. 承認ボタンを押下

### 5. 履歴管理

#### 履歴閲覧

1. メイン画面の履歴表示領域で履歴一覧を確認
2. 再表示ボタンを押下で過去の分析結果を表示

#### 履歴削除

1. 削除したい履歴の削除ボタンを押下
2. 確認ダイアログで削除を確認

#### 案件URL/メモ編集

1. 編集したい履歴のURL/メモ編集ボタンを押下
2. 案件URLおよび案件メモを編集
3. 保存ボタンを押下

### 6. 応募予定案件管理

#### 応募予定案件登録

1. 分析結果表示後、応募予定案件登録ボタンを押下
2. 応募予定案件一覧に追加

#### 応募予定案件確認

1. メイン画面の応募予定案件表示領域で一覧を確認
2. 詳細表示ボタンを押下で案件詳細を表示

#### 応募予定案件削除

1. 削除したい案件の削除ボタンを押下
2. 確認ダイアログで削除を確認

## APIキー取得ガイド内容

### OpenAI APIキー取得方法

#### 手順1：OpenAIアカウント作成

1. OpenAI公式サイト（https://openai.com/）へアクセス
2. Sign Upボタンを押下
3. メールアドレスとパスワードを入力
4. メール認証を完了

#### 手順2：APIキー発行

1. OpenAIアカウントへログイン
2. API Keysページへアクセス（https://platform.openai.com/api-keys）
3. Create new secret keyボタンを押下
4. APIキー名を入力
5. Createボタンを押下
6. 表示されたAPIキーをコピー

#### 手順3：AI Job InsightへAPIキー設定

1. AI Job Insightのメイン画面へ戻る
2. APIキー設定領域でAIサービス種別「OpenAI」を選択
3. コピーしたAPIキーを入力
4. 保存ボタンを押下

#### 注意事項

- APIキーは第三者に公開しないでください
- APIキーは安全に管理してください
- OpenAI APIの利用には料金が発生します

### Google Gemini APIキー取得方法

#### 手順1：Google AI Studioへアクセス

1. Google AI Studio（https://aistudio.google.com/）へアクセス
2. Googleアカウントでログイン

#### 手順2：APIキー発行

1. Get API Keyボタンを押下
2. Create API key in new projectを選択
3. 表示されたAPIキーをコピー

#### 手順3：AI Job InsightへAPIキー設定

1. AI Job Insightのメイン画面へ戻る
2. APIキー設定領域でAIサービス種別「Google Gemini」を選択
3. コピーしたAPIキーを入力
4. 保存ボタンを押下

#### 注意事項

- APIキーは第三者に公開しないでください
- APIキーは安全に管理してください
- Google Gemini APIの利用には制限があります
