# AI Job Insight 詳細設計書 - 応募予定案件管理機能

---

# 1. 機能概要

## 1.1 機能名

応募予定案件管理機能

## 1.2 機能ID

FUNC-APPLICATION-001

## 1.3 機能の目的

応募すると決めた案件をリストとして保存・管理する。一覧表示、案件内容の確認、関連する分析結果や投稿文へのアクセス、削除を行う。

## 1.4 利用される画面

- メイン画面 応募予定案件表示領域

---

# 2. 処理の実現方式

この機能は以下の技術・方式で実現する。

- フロントエンド：React
- 言語：TypeScript
- データ保存：localStorage
- データ形式：JSON

---

|項目名|型|必須|説明|
|---|---|---|---|
|案件情報|object|○|応募予定の案件情報|

# 4. 出力データ

|項目名|型|説明|
|---|---|---|
|応募予定案件一覧|array|保存済み応募予定案件の配列|
|案件詳細|object|選択した案件の詳細データ|

---

# 5. 処理の流れ

## 5.1 応募予定案件登録

1. ユーザーが分析結果表示画面で応募予定案件登録ボタンを押下
2. 応募予定案件データオブジェクトを作成
   - 案件ID（UUID）
   - 案件情報
   - 履歴ID
   - 投稿文
   - 登録日時
3. localStorageへ保存
4. 応募予定案件一覧を更新
5. 成功メッセージを表示

## 5.2 応募予定案件一覧表示

1. localStorageから応募予定案件データを取得
2. 登録日時の降順でソート
3. 応募予定案件一覧をレンダリング
4. 各案件の表示項目
   - 登録日時
   - 案件説明（先頭100文字）
   - 報酬
   - 案件URL（リンク表示）
   - 詳細表示ボタン
   - 削除ボタン

## 5.3 案件詳細表示

1. ユーザーが詳細表示ボタンを押下
2. 案件IDを取得
3. localStorageから該当案件を取得
4. 案件詳細ダイアログを表示
5. 表示項目
   - 案件情報
   - 案件URL
   - 案件メモ
   - 投稿文
   - 関連する分析結果へのリンク

## 5.4 応募予定案件削除

1. ユーザーが削除ボタンを押下
2. 確認ダイアログを表示
3. ユーザーが削除を確認
4. 案件IDを取得
5. localStorageから該当案件を削除
6. 応募予定案件一覧を更新

## 5.5 関連分析結果へのアクセス

1. ユーザーが案件詳細ダイアログで分析結果リンクを押下
2. 履歴IDを取得
3. 履歴管理機能の履歴再表示処理を呼び出し
4. 案件詳細ダイアログを閉じる
5. 分析結果表示領域へ遷移

# 8. エラー処理

|エラー内容|対応|
|---|---|
|案件登録失敗|エラーメッセージ「案件の登録に失敗しました」を表示|
|案件取得失敗|エラーメッセージ「案件の取得に失敗しました」を表示|
|案件削除失敗|エラーメッセージ「案件の削除に失敗しました」を表示|
|関連履歴取得失敗|エラーメッセージ「関連する分析結果が見つかりませんでした」を表示|

# 10. 制約事項

- 応募予定案件数の上限は設けない（localStorageの容量制限に依存）
- 案件IDはUUID形式
- 削除した案件は復元できない
- 応募予定案件は登録日時の降順で表示

# 12. 備考

## 応募予定案件データ構造

```json
{
  "id": "uuid",
  "jobInfo": {
    "description": "案件説明",
    "workContent": "仕事内容",
    "conditions": "条件",
    "reward": "報酬",
    "url": "案件URL",
    "memo": "案件メモ"
  },
  "historyId": "履歴ID",
  "proposal": "投稿文",
  "registeredAt": "2026-03-12T10:00:00Z"
}
```

## 応募予定案件一覧表示項目

- 登録日時：YYYY年MM月DD日 HH:mm 形式
- 案件説明：先頭100文字（超える場合は「...」を追加）
- 報酬：報酬情報
- 案件URL：リンク表示（URLが設定されている場合）
- 詳細表示ボタン
- 削除ボタン

## 案件詳細ダイアログ

### 表示項目

- 案件説明
- 仕事内容
- 条件
- 報酬
- 案件URL（リンク表示）
- 案件メモ
- 投稿文
- 関連する分析結果へのリンク（履歴IDが設定されている場合）

### ボタン

- 閉じるボタン

## 応募予定案件削除確認ダイアログ

- メッセージ：「この案件を削除してもよろしいですか？」
- 削除ボタン
- キャンセルボタン

## 応募予定案件登録ボタンの配置

- 分析結果表示領域の下部に配置
- ボタンラベル：「応募予定案件として登録」
