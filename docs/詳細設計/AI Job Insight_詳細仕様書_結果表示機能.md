# AI Job Insight 詳細設計書 - 結果表示機能

---

# 1. 機能概要

## 1.1 機能名

結果表示機能

## 1.2 機能ID

FUNC-RESULT-001

## 1.3 機能の目的

AI分析結果を画面へ表示し、ユーザーが案件への応募判断を行うための情報を提供する。スキル適合度をレーダーチャートで可視化し、直感的な理解を支援する。

## 1.4 利用される画面

- メイン画面 分析結果表示領域
- メイン画面 対話支援領域

---

# 2. 処理の実現方式

この機能は以下の技術・方式で実現する。

- フロントエンド：React
- 言語：TypeScript
- UI表示：React Component
- チャート表示：Chart.js または Recharts
- スタイリング：CSS / Tailwind CSS

---

# 3. 入力データ

|項目名|型|必須|説明|
|---|---|---|---|
|おすすめ度|number|○|1〜5の星評価|
|良い点|string|○|案件の良い点|
|注意点|string|○|案件の注意点|
|スキル適合度データ|object|○|レーダーチャート表示用データ|
|応募戦略|string|○|応募する場合の戦略|
|強調すべきスキル|string|○|強調すると良いスキル|
|提案ポイント|string|○|提案時のポイント|
|応答促進メッセージ|string|○|追加対話を促す案内文|

---

# 4. 出力データ

|項目名|型|説明|
|---|---|---|
|表示用HTML|ReactElement|レンダリングされた結果表示要素|

---

# 5. 処理の流れ

1. 分析結果データの受け取り
2. データ整合性チェック
3. 応募おすすめ度の星表示レンダリング
4. 案件評価理由の表示
5. 良い点セクションの表示
6. 注意点セクションの表示
7. スキル適合度レーダーチャートの描画
8. 応募戦略セクションの表示
9. 強調すべきスキルの表示
10. 提案ポイントの表示
11. 応答促進メッセージの表示
12. 対話支援領域の表示準備
13. 画面スクロール調整

---

# 6. 使用するデータ

|データ名|保存場所|説明|
|---|---|---|
|分析結果|コンポーネントstate|受け取った分析結果データ|
|スキル項目ラベル|localStorage|レーダーチャートの軸ラベル|

---

# 7. AIプロンプト仕様

なし

---

# 8. エラー処理

|エラー内容|対応|
|---|---|
|データ不整合|エラーメッセージ「分析結果の表示に失敗しました」を表示|
|チャートレンダリング失敗|スキル適合度セクションを非表示にし、テキストで表示|
|表示要素の取得失敗|エラーメッセージ「表示処理に失敗しました」を表示|

---

# 10. 制約事項

- レーダーチャートはスキル項目ラベルの数に応じて軸数が変動
- スキル項目ラベルが未設定の場合はデフォルト値を使用
- 星評価は1〜5の整数のみ対応
- レーダーチャートの値は0〜10の範囲

# 12. 備考

## 応募おすすめ度表示仕様

- 表示形式：★マークによる5段階評価
- 例：★★★★☆（4の場合）

## スキル適合度レーダーチャート仕様

- 表示形式：レーダーチャート
- 軸：ユーザーが設定したスキル項目ラベル
- デフォルト軸：文書作成、Web構築、プログラミング、仕様検討、イラスト作成
- 値の範囲：0〜10
- 色：プライマリカラー

## 応答促進メッセージ

分析結果の末尾に固定で表示する。

表示例：

```
質問はありますか？
追加で情報をいただければ、一緒に投稿文を作成することもできます。
```

## セクション構成

1. 応募おすすめ度
2. 案件評価理由
   - 良い点
   - 注意点
3. スキル適合度（レーダーチャート）
4. 応募戦略
   - 強調すべきスキル
   - 提案ポイント
5. 応答促進メッセージ
6. 対話支援領域
