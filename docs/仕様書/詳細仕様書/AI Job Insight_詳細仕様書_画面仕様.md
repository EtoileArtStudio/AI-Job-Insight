## 全体UIデザインコンセプト

### デザインテーマ

- **ビジネスユース向けの清潔感のあるデザイン**
- **白と青を基調としたライトテーマ**
- クリーンで見やすく、プロフェッショナルな印象を重視
- モダンで洗練されたインターフェース

### カラーパレット

- **ベース色**：白（#FFFFFF）
- **プライマリカラー**：青系（#3B82F6 - Tailwind CSS blue-500相当）
- **背景色**：薄いグレーブルー（#F1F5F9 - Tailwind CSS slate-100相当）
- **アクセントカラー**：明るい青（#60A5FA - Tailwind CSS blue-400相当）
- **テキストカラー**：濃いグレー（#1E293B - Tailwind CSS slate-800相当）
- **エラーカラー**：赤系（#EF4444）
- **成功カラー**：緑系（#10B981）

### レイアウト方針

- **サイドナビゲーション方式**を採用
- 左側に固定ナビゲーションバー（約200-250px幅）
- メインコンテンツエリアは中央から右側に配置
- 余白を適切に確保し、情報の視認性を向上
- カード形式のコンポーネント配置で視覚的な階層を明確化

### UIコンポーネントの特徴

- **カード型UI**：影（シャドウ）を活用した立体的な表現
- **角丸デザイン**：柔らかさと親しみやすさを演出
- **ボタンデザイン**：明確な視覚的フィードバック（ホバー時の色変化）
- **入力フィールド**：境界線とフォーカス状態の明確化
- **タグUI**：スキル項目などを視覚的に区別（色付き、角丸、削除可能）
- **アイコン使用**：直感的な操作を支援
- **スムーズなアニメーション**：ページ遷移やコンポーネント表示時の自然な動き

## メイン画面構成

メイン画面は以下の領域で構成される。

### 全体レイアウト構造

#### 左サイドナビゲーションバー

**外観**
- 幅：固定幅（250px）
- 背景色：ダークグレー（#2c3e50）
- テキスト色：白（#FFFFFF）
- 位置：固定（position: fixed）
- 高さ：画面全体（100vh）

**ナビゲーション項目（作業フロー順）**
1. プロフィール（👤アイコン）
2. 案件分析（📊アイコン）
3. 応募文章作成（✍️アイコン）
4. 設定（⚙️アイコン）

**想定作業フロー**
1. プロフィール設定（初回のみ）
2. 案件情報入力とAI分析
3. 分析結果確認
4. 「この案件の応募文章を作成」ボタンで遷移
5. 応募文章作成（案件情報連動）

**特徴**
- アクティブ項目は背景色が変化（#34495e）、左側に青いボーダー（#3498db）
- ホバー時に背景色が変化（#34495e）
- アイコン + テキストラベルの組み合わせ
- React RouterのNavLinkを使用した画面遷移

#### メインコンテンツエリア

**外観**
- 背景色：薄いグレー（#f5f6fa）
- 左マージン：250px（サイドバー幅分）
- パディング：上下左右に適度な余白（2rem）
- スクロール可能

**レイアウト方式**
- 各ページごとに異なるレイアウトを採用
- セクションごとにCard UIで区切り
- ページタイトルは共通スタイル（1.8rem、#2c3e50）
