# AI Job Insight 詳細設計書 - 履歴管理機能

---

# 1. 機能概要

## 1.1 機能名

履歴管理機能

## 1.2 機能ID

FUNC-HISTORY-001

## 1.3 機能の目的

過去の案件分析結果を保存・参照・編集・削除する。ユーザーが過去の分析内容を再確認し、案件URLやメモを編集できるようにする。

## 1.4 利用される画面

- メイン画面 履歴表示領域

---

# 2. 処理の実現方式

この機能は以下の技術・方式で実現する。

- フロントエンド：React
- 言語：TypeScript
- データ保存：localStorage
- データ形式：JSON

---

# 3. 入力データ

|項目名|型|必須|説明|
|---|---|---|---|
|案件情報|object|○|ユーザーが入力した案件情報|
|案件URL|string|任意|案件URL|
|案件メモ|string|任意|ユーザーのメモ|
|分析結果|object|○|AI分析結果|
|分析日時|string|○|分析を実行した日時|

---

# 4. 出力データ

|項目名|型|説明|
|---|---|---|
|履歴一覧|array|保存済み履歴の配列|
|履歴詳細|object|選択した履歴の詳細データ|

---

# 5. 処理の流れ

## 5.1 履歴保存

1. 分析実行完了時に自動的に履歴保存処理を開始
2. 履歴データオブジェクトを作成
   - 履歴ID（UUID）
   - 案件情報
   - 案件URL
   - 案件メモ
   - 分析結果
   - 分析日時
3. localStorageへ保存
4. 履歴一覧を更新

## 5.2 履歴一覧表示

1. localStorageから履歴データを取得
2. 分析日時の降順でソート
3. 履歴一覧をレンダリング
4. 各履歴の表示項目
   - 分析日時
   - 案件説明（先頭100文字）
   - 応募おすすめ度
   - 再表示ボタン
   - 削除ボタン
   - URL/メモ編集ボタン

## 5.3 履歴再表示

1. ユーザーが再表示ボタンを押下
2. 履歴IDを取得
3. localStorageから該当履歴を取得
4. 案件入力領域へ案件情報を表示
5. 分析結果表示領域へ分析結果を表示
6. 画面をスクロールして分析結果へ移動

## 5.4 履歴削除

1. ユーザーが削除ボタンを押下
2. 確認ダイアログを表示
3. ユーザーが削除を確認
4. 履歴IDを取得
5. localStorageから該当履歴を削除
6. 履歴一覧を更新

## 5.5 案件URL/メモ編集

1. ユーザーがURL/メモ編集ボタンを押下
2. 履歴IDを取得
3. localStorageから該当履歴を取得
4. 編集ダイアログを表示
5. 案件URLおよび案件メモを編集可能状態で表示
6. ユーザーが内容を編集
7. 保存ボタンを押下
8. localStorageへ上書き保存
9. 履歴一覧を更新

# 8. エラー処理

|エラー内容|対応|
|---|---|
|履歴保存失敗|エラーメッセージ「履歴の保存に失敗しました」を表示|
|履歴取得失敗|エラーメッセージ「履歴の取得に失敗しました」を表示|
|履歴削除失敗|エラーメッセージ「履歴の削除に失敗しました」を表示|
|履歴更新失敗|エラーメッセージ「履歴の更新に失敗しました」を表示|

---

# 10. 制約事項

- 履歴数の上限は設けない（localStorageの容量制限に依存）
- 履歴IDはUUID形式
- 削除した履歴は復元できない
- 履歴は分析日時の降順で表示

```json
{
  "id": "uuid",
  "jobInfo": {
    "description": "案件説明",
    "workContent": "仕事内容",
    "conditions": "条件",
    "reward": "報酬",
    "url": "案件URL",
    "memo": "案件メモ"
  },
  "analysisResult": {
    "score": 4,
    "goodPoints": "案件の良い点",
    "concerns": "案件の注意点",
    "skillMatch": {
      "文書作成": 8,
      "Web構築": 6,
      "プログラミング": 7,
      "仕様検討": 5,
      "イラスト作成": 2
    },
    "strategy": "応募戦略",
    "keySkills": "強調すべきスキル",
    "proposalPoints": "提案ポイント"
  },
  "analyzedAt": "2026-03-12T10:00:00Z"

```

## 履歴一覧表示項目

- 分析日時：YYYY年MM月DD日 HH:mm 形式
- 案件説明：先頭100文字（超える場合は「...」を追加）
- 応募おすすめ度：★マーク表示
- 再表示ボタン
- 削除ボタン
- URL/メモ編集ボタン

## 案件URL/メモ編集ダイアログ

- 案件URLフィールド（テキスト入力）
- 案件メモフィールド（複数行テキスト入力）
- 保存ボタン
- キャンセルボタン

## 履歴削除確認ダイアログ

- メッセージ：「この履歴を削除してもよろしいですか？」
- 削除ボタン
- キャンセルボタン
